如何设置 DoH？
1. 在上方找到有效的 DoH，点击「复制」按钮
2. 打开浏览器设置，找到 DoH / 安全 DNS 选项
3. 选择「自定义」，粘贴复制的 DoH URL

各浏览器设置位置：
Firefox: 设置 → 隐私与安全 → DNS over HTTPS → 自定义 → 粘贴 URL
Chrome: 设置 → 隐私和安全 → 安全 → 使用安全 DNS → 自定义 → 粘贴 URL
Edge: 设置 → 隐私、搜索和服务 → 安全 → 使用安全 DNS → 自定义 → 粘贴 URL
Safari: Safari 不支持原生 DoH，需安装 DNSecure 等 DoH 管理类 App 来启用
Android: 安卓系统不支持原生 DoH，推荐使用 狐猴浏览器 等支持 DoH 的浏览器，在浏览器设置中配置

测试不正常？
如果设置 DoH 后网站打不开或加载很慢：
• 检查浏览器是否设置了无效或不稳定的 DoH
• 回到本工具「DoH 检测」页重测，确认 DoH 状态为「有效」
• 换一个延迟更低的 DoH 提供商（延迟 < 100ms 最佳）
• 暂时关闭浏览器 DoH，确认是否能恢复访问
• 部分网络环境会拦截 DoH 流量，尝试换网络或使用代理

测试不正常怎么办？
如果网站不可访问或加载缓慢，可能是浏览器设置了无效或不稳定的 DoH：
• 先到「DoH 检测」页测试你的 DoH 是否仍然有效
• 如果 DoH 已失效，更换一个有效的 DoH 提供商
• 临时关闭浏览器 DoH（设置为「使用系统 DNS」），确认网站是否恢复
• 关闭后恢复 = DoH 问题；关闭后仍不行 = 网络或网站本身问题
• 部分公司/学校网络会拦截 DoH，可尝试更换网络环境
